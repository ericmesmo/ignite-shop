## Run Locally

Clone the project

```bash
  git clone https://github.com/ericmesmo/ignite-shop
```

Go to the project directory

```bash
  cd ignite-shop
```

Install dependencies

```bash
  pnpm install
```

Start the server

```bash
  pnpm dev
```
